## 📂 Dataset

The dataset used in this project is publicly available on Kaggle.

Due to GitHub file size limitations, it is not included in this repository.

You can access the dataset here:
https://www.kaggle.com/code/gpreda/credit-card-fraud-detection-predictive-models?select=creditcard.csv

After downloading, place it in the `data/` folder before running the model.
